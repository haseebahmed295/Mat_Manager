# MatShelf

A visual material shelf and organization addon for Blender.

## Features
- **Visual Grid Gallery**: Real-time thumbnail preview grid in the 3D Viewport sidebar (`N` panel).
- **Search & Filters**: Search on-type and filter by categories.
- **Category Manager**: Add, remove, and rename categories.
- **Batch Assign**: Apply materials to all selected objects.
- **Operations Menu (`▼`)**: Rename, Select Users, Duplicate, Delete, and Set Category.
- **Cleanup Utilities**: Merge duplicate materials (`.001`, `.002`) and Purge unused data.
- **Gallery Settings (`⚙`)**: Configure columns, thumbnail size, and preview shape (saved in scene).

## Installation
1. Zip the `Mat_Manager` folder (containing `__init__.py`, `blender_manifest.toml`, `properties.py`, `operators.py`, `ui.py`, and `helpers.py`).
2. In Blender, go to **Edit > Preferences > Get Extensions**.
3. Click the cog menu at the top-right and select **Install from Disk...**.
4. Select the zip file and click install.

## Usage
1. Open the **3D Viewport**.
2. Press `N` to toggle the Sidebar.
3. Open the **MatShelf** tab.
